## 📐 Dimensiones y Espaciado

### Máximo Ancho de Contenido
```
- Desktop:   1280px (max-w-7xl)
- Tablet:    768px  (md:)
- Mobile:    100%   (sin padding)
```

### Padding y Márgenes
```
- Secciones principales:  py-24 (96px top+bottom)
- Contenedores internos:  px-8  (32px left+right)
- Elementos:              p-8   (32px all)
```

### Tipografía
```
H1/Hero:     text-7xl   md:text-5xl    (tracking-tighter)
H2/Section:  text-5xl   md:text-4xl    (font-bold)
H3/Card:     text-lg    (font-bold)
Párrafos:    text-lg    text-xl        (font-light)
Labels:      text-xs    (uppercase)
```

## 🎬 Efectos de Interacción

### Hover States
```javascript
// Botones
.hover:scale-105        // Crece 5%
.transition-transform   // Transición suave (300ms)

// Tarjetas (módulos)
.hover:bg-surface-bright // Fondo más claro

// Links
.hover:text-primary     // Cambia a color primary
.transition-colors      // Transición suave
```

### Animaciones
```javascript
.animate-pulse          // Pulso continuo (indicador de activo)
.animate-ping           // Ping (notificación)
```

### Gradientes
```javascript
// Texto gradient
background: linear-gradient(135deg, #ff8f6f 0%, #ff7851 100%)
-webkit-background-clip: text
-webkit-text-fill-color: transparent

// Botones CTA
background: linear-gradient(135deg, #ff8f6f 0%, #ff7851 100%)
```

## 📱 Responsive Breakpoints

### Mobile First (< 640px)
```
- Stack vertical
- Full width
- Botones más grandes
- Iconos más grandes
```

### Tablet (640px - 1024px)
```
- Grid 2 columnas
- Espacios reducidos
- Font sizes aumentan
```

### Desktop (> 1024px)
```
- Grid 4 columnas
- Espacios amplios
- Layouts asimétricas
```

---

## 🔍 Orden de Render

```
1. Header (fixed) - siempre visible
2. Hero - atrae atención
3. Problem - Identifica dolor
4. System - Ofrece solución
5. Author - Credibilidad
6. Social Proof - Validación social
7. Offer - Call to action
8. Footer - Cierre
```

---

## 💡 Principios de Diseño Aplicados

✨ **F-Pattern**: Los ojos recorren en forma de F
✨ **AIDA**: Atención → Interés → Deseo → Acción
✨ **Scarcity**: Precio con descuento, límite implícito
✨ **Social Proof**: Testimonios y números
✨ **Trust Signals**: Seguridad, métodos de pago

---

**Esta estructura visual se traduce directamente en los componentes React con Tailwind CSS.** 🎨
